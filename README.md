# Monorepo with Go Backend and React Frontend

This is a monorepo containing a Go backend and a React frontend. The entire application can be run using Docker Compose.

## Prerequisites

* Docker
* Docker Compose

## Getting Started

To get the application running, follow these steps:

1.  Clone the repository.
2.  Navigate to the root of the repository.
3.  Run the following command:

    ```bash
    docker-compose up -d
    ```

The backend will be available at `http://localhost:8080` and the frontend will be available at `http://localhost:5173`.
